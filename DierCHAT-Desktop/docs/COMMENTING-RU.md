# Стиль комментариев (русский)

Перед **логическим блоком** (не перед каждой строкой):

```ts
// ==================== БЛОК: КРАТКОЕ ИМЯ ====================
// Назначение: что делает блок.
// Связь: какие экраны / API задействованы.
```

Старые комментарии при правке файла можно заменять на такой формат; массовая зачистка всего репозитория — поэтапно по модулям.

# ТЗ §25 — Мультивыбор сообщений

## Реализовано
- **Долгое нажатие** (~480 ms) на пузырёк — вход в режим и выбор сообщения.
- **Клик** по сообщению в режиме — вкл/выкл выделения.
- **Shift + клик** — выделение диапазона между последним якорем и текущим (порядок по времени в чате).
- **Контекстное меню** — пункт «Выбрать».
- **Верхняя панель**: счётчик, Удалить (только свои), Переслать, Копировать, Избранное, Закрепить и Ответить (оба только при **одном** выделении), отмена (✕).
- **Escape** — выход из режима.
- **Смена чата** — сброс выделения.
- **Лимит** — `MAX_MESSAGE_SELECTION = 100` (`src/lib/messageSelection.ts`).
- **Системные** сообщения (`type === 6`) и удалённые — не выделяются.
- **Пересылка** нескольких — последовательные вызовы API для каждого id.

## Файлы
- `src/lib/messageSelection.ts` — правила выбора и текст для копирования.
- `src/components/chat/MessageBubble.tsx` — UI чекбоксов, long-press, пункт меню.
- `src/components/chat/MessageList.tsx` — прокидывание пропсов.
- `src/components/chat/ChatView.tsx` — состояние и панель действий.
